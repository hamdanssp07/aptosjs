# Getting Started

## Quick setup

Install AptosJS

```bash
npm install aptosjs aptos
```

You are ready to go 

## Hooks 

### useAccount
Hook for returning details on a specific Aptos account.

### useBalance
Hook for accessing user balance.

### useCollection
Hook for returning specific information on an Aptos NFT collection.

### useGas
Hook for returning the current gas price.

### useNFT
Hook for returning data on a specific Token (NFT).

### useTransaction

Hook for returning data on a specific transaction.

### useAns

Hook for accessing Aptos Ans data.

## Send

### sendFaucet

Send 1 APT to a devnet or testnet account.
